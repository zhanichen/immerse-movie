## 此專案使用哪些技術
* pug、sass
* vue.js實作開關視窗、關鍵字搜尋、類型篩選
* TweenMax實作scroll滾動效果
* Boostrap RWD
* CSS RWD



##### 備註：此專案為吳哲宇課程電影介面實作練習作品之延伸作品
